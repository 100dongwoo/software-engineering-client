# 소프트웨어 공학 프로젝트 클라이언트 
### 소개
웹서비스 소프트웨어입니다. <br/>
팀: 최세홍(조장), 김재훈, 백동우
#### 기간
2020.09.11 ~ ing

## 실행 방법 
```
# 깃허브에서 소스코드를 다운로드 받습니다.

git clone git@github.com:100dongwoo/software-engineering-client.git

# 프로젝트의 기존 패키지를 설치합니다.

yarn install

# 로컬 서버를 통해 프로젝트를 시작합니다.

yarn start

```
##### 사용한 라이브러리
- react-router-dom
- styled-components
- Material UI (UI 컴포넌트 라이브러리)
- useFormik, yup
- Context
- Infinite Scroll
- moment
- react-portal-tooltip
